## 🛜 Port-Scanning-with-Zenmap 🛜

I developed a port scanning project utilizing Zenmap. Zenmap or a nmap (Network Mapper) tool, is an open-source tool that allows administrators to learn whether the device is turned on, which services are running on the open ports, and which operating system the device is using by sending packets to the device and ports of the device on the network, and looking at the responses. In conclusion, the Zenmap port scanning project has helped me acquire hands-on experience in firewall detection and vulnerability scanning.
